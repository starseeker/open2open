# Pre-built OCCT 7.9.3 static libraries (linux-x86_64 / Release)

This directory contains a pre-built snapshot of
[OpenCASCADE Technology 7.9.3](https://dev.opencascade.org/) for
**linux x86_64**, built with GCC in Release mode, as static libraries.
It is committed here so that subsequent CI sessions and developer checkouts do
not need to recompile OCCT (which takes ~15–20 minutes).

## Structure

```
build_artifacts/occt/
├── include/opencascade/   ← flat header tree (generated by cmake configure)
├── lib/
│   ├── libTK*.a           ← static libraries
│   └── cmake/
│       └── opencascade-7.9.3/
│           ├── OpenCASCADEConfig.cmake
│           ├── OpenCASCADETargets.cmake          ← relocatable (uses relative paths)
│           └── OpenCASCADE*Targets.cmake         ← per-component aliases
```

## Build flags used

```
cmake OCCT-7_9_3/ \
  -DCMAKE_BUILD_TYPE=Release \
  -DBUILD_MODULE_Draw=OFF \
  -DBUILD_MODULE_Visualization=OFF \
  -DBUILD_MODULE_ApplicationFramework=OFF \
  -DBUILD_MODULE_DataExchange=OFF \
  -DBUILD_MODULE_Inspection=OFF \
  -DBUILD_LIBRARY_TYPE=Static \
  -DUSE_FREETYPE=OFF \
  -DUSE_FREEIMAGE=OFF
```

## Usage

`open2open/CMakeLists.txt` automatically detects this directory as a fallback
if no `OpenCASCADE_DIR` is set and no system OCCT package is found.  No extra
cmake flags are needed:

```bash
mkdir build && cd build
cmake ..   # picks up build_artifacts/occt/ automatically
make -j$(nproc)
```

To override and use a different OCCT installation:

```bash
cmake .. -DOpenCASCADE_DIR=/path/to/your/occt/lib/cmake/opencascade-x.y.z
```

## Refreshing

If you need to rebuild (e.g. for a different architecture, compiler, or OCCT
version), follow the build instructions in `.github/copilot-instructions.md`,
then copy the outputs:

```bash
cp -r /tmp/occt_build/include/opencascade/. build_artifacts/occt/include/opencascade/
cp /tmp/occt_build/lin64/gcc/lib/libTK*.a   build_artifacts/occt/lib/

# Generate a relocatable OpenCASCADETargets.cmake
python3 - << 'EOF'
import re, shutil
src  = '/tmp/occt_build/OpenCASCADETargets.cmake'
dst  = 'build_artifacts/occt/lib/cmake/opencascade-7.9.3/OpenCASCADETargets.cmake'
with open(src) as f: content = f.read()
content = content.replace(
    'cmake_policy(VERSION 2.8.3...3.29)',
    'cmake_policy(VERSION 2.8.3...3.29)\n\n'
    'get_filename_component(_OCCT_SELF_DIR "${CMAKE_CURRENT_LIST_FILE}" PATH)\n'
    'get_filename_component(_OCCT_INSTALL_PREFIX "${_OCCT_SELF_DIR}" PATH)\n'
    'get_filename_component(_OCCT_INSTALL_PREFIX "${_OCCT_INSTALL_PREFIX}" PATH)\n'
    'get_filename_component(_OCCT_INSTALL_PREFIX "${_OCCT_INSTALL_PREFIX}" PATH)\n')
content = re.sub(r'"/tmp/[^"]+/lib/(libTK[^"]+\.a)"',
                 r'"${_OCCT_INSTALL_PREFIX}/lib/\1"', content)
with open(dst, 'w') as f: f.write(content)
EOF

# Copy per-component aliases (they are identical to OpenCASCADETargets.cmake)
for c in FoundationClasses ModelingData ModelingAlgorithms DETools; do
  cp build_artifacts/occt/lib/cmake/opencascade-7.9.3/OpenCASCADETargets.cmake \
     build_artifacts/occt/lib/cmake/opencascade-7.9.3/OpenCASCADE${c}Targets.cmake
done
cp /tmp/occt_build/OpenCASCADEConfig.cmake \
   build_artifacts/occt/lib/cmake/opencascade-7.9.3/
```
